Hyperskill - Java Backend Developer track's four graduate projects

Code Sharing

  • This project uses Freemarker to render front-end.
  • Code snippet that doesn't expired or has a restricted number of views can be access using route /code.
  • Code snippet that has number of views restriction or time expiration will be accessible at /code/{id}.
  • Data is persisted in a local H2 database.
Hyperskill's page Github's repository

Recipe

  • Registration is public.
  • Authentication uses username and password.
  • Password is encrypted in the database.
  • Only authenticated user can add and view recipes.
  • Only owner of the recipe can delete and update a recipe.
  • Authenticated users can search for recipes using either name or category.
  • Data is persisted in a local H2 database.
Hyperskill's page Github's repository

Web Quiz Engine

  • Registraion is public.
  • Authentication uses username and password.
  • Password is encrypted in the database.
  • Only authenticated user can send request to the quiz engine.
  • Only owner of the recipe can delete the quiz.
  • The requests' results can be queried with size and pageSize parameters. Deault pageSize is 10.
  • Data is persited in a local H2 database.

Car Sharing

  • This project is the most basic out of the four.
  • It is a command line program.
  • Data is persisted in H2 database.
  • The aim of this project is practice using Spring JDBC and implementing Data Access Object pattern.
